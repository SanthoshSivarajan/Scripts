# Active Directory Scripts

## Overview

This repository contains a collection of PowerShell scripts focused on Active Directory administration, automation, and operational tasks.

These scripts are designed to support day-to-day administrative activities, improve efficiency, and provide practical solutions to common challenges encountered in enterprise environments.

---

## Note

These scripts were previously published across my blog:

https://portal.sivarajan.com/

This repository represents an effort to consolidate, preserve, and make these scripts more accessible through GitHub. As part of this migration, scripts may be updated, reorganized, and improved over time.

---

## Purpose

The goal of this repository is to provide:

- Practical and reusable Active Directory scripts  
- Solutions to real-world administrative and operational scenarios  
- Reference implementations for common identity and directory tasks  
- A centralized location for previously published technical content  

---

## Scope

The scripts in this repository cover areas such as:

- User and group management  
- Computer and system administration  
- Domain and directory queries  
- Service and configuration management  
- Reporting and data extraction  

---

## Usage

- Scripts are provided as-is and may require modification based on your environment  
- Always review and test scripts in a non-production environment before use  
- Ensure appropriate permissions are in place before executing any script
